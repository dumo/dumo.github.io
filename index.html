Safari使用自定义PAC

发表于 2016-05-07

1.根据gfwlisit自动生成pac文件，在这里https://github.com/zhiyi7/gfw-pac 下载pac，也可以使用它制作自己的pac文件。
如果使用作者提供的pac，记得要修改成自己的代理地址：
例如：

var proxy = "PROXY 127.0.0.1:8888";

2.MAC现在已经不能使用本地的pac文件,必须通过网络访问PAC文件。可以将proxy.pac放在网络上,比如自己的VPS上。也可以开启本地web服务器,将其放在本地。
应该在web服务器上将pac文件的MIME类型设置为application/x-ns-proxy-autoconfig 或者 application/x-javascript-config,当然不设置也没有关系。
如果使用nginx,则在/etc/nginx/mime.types文件中添加即可。
启用本地apache服务:

#apachectl start

然后将proxy.pac放入/Library/WebServer/Documens/目录下。

配置PAC代理时,填入http://localhost/proxy.pac即可。

参考：
MAC平台PAC文件编写和自动代理配置
